- 背景：北京即设科技（即时设计）图形编辑器/白板、平台分层重构、复杂功能迭代与性能治理

<!-- 分组说明：强调你的“怎么用、解决了什么问题” -->
- 前端与框架
  - JavaScript / TypeScript：类型驱动的可维护代码与设计模式实践
  - React / Next.js：SSR/ISR、路由与数据流，支撑复杂编辑器/白板场景
  - Monaco Editor / React Flow：可视化编辑与流程编排，提升搭建效率与可视化能力
  - Radix UI + Tailwind CSS：无障碍与主题化组件库建设，统一交互与视觉规范

- 工程化与效率
  - Monorepo（Turborepo）：多包协作与依赖统一，构建加速与版本治理
  - 构建优化：Vite / Webpack 的缓存、按需、分包与性能度量
  - 质量保障：Jest / Vitest 单元测试、严格 Code Review 规则
  - CI/CD：GitHub Actions / 自建流水线，自动发布、质量门禁与灰度策略

- 全栈与部署
  - Prisma + PostgreSQL：类型安全 ORM、迁移与查询性能优化
  - Supabase：认证、存储、实时能力，快速交付基础后端
  - Vercel：前端与全栈托管（Edge / Serverless），提升交付速度
  - Docker / PM2：容器化与进程守护，可观测与回滚保障

- AI 工程
  - Dify Workflow：需求到代码生成的工作流编排，构建自然语言→代码闭环
  - RAG + 向量数据库：上下文检索增强，提高生成与改写的准确性
  - Multi‑Agent / Prompt Engineering：复杂任务拆解、指令优化与输出调优

- 图形与渲染
  - CanvasKit / Skia：底层渲染能力封装，矩阵变换与 Shader 实战
  - 复杂交互：选中 / 移动 / 缩放 / 模式切换等核心行为的性能与一致性
  - Electron / React Native：桌面端与移动端 0→1 交付

- 协同与数据结构
  - CRDT（Fraction Indexing）：层级冲突处理与多用户一致性保障
  - Undo / Redo 历史栈：可恢复性与响应性能优化
  - 文件解析与序列化：Figma / Sketch → Kiwi Schema（Protocol Buffer），含加密与跨平台兼容
